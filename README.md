# 🎯 개발 환경

- Xcode 13.0+
- Swift 5.5+
- iOS 14.5+
 
<br/> 

# 📝 프로젝트 목표
```
1. 기존 올라 앱보다 더 많은 산 데이터를 제공
2. 기존 올라 앱보다 뛰어난 접근성 제공
3. CoreData, CoreLocation, CoreMotion, MapKit을 활용한 앱
4. 외부 라이브러리에 대한 의존성이 없는 앱
```

<br/>

# 💬 프로젝트 소개 

산타는 등산 기록을 측정하고, 측정된 데이터를 사용자가 쉽게 파악할 수 있도록 해주는 iOS 어플리케이션 입니다.
